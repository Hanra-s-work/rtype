Detailed Description

(Deprecated: Use executor_work_guard.) Class to inform the io_context when it has work to do. The work class is used to inform the io_context when work starts and finishes. This ensures that the io_context object's run() function will not exit while work is underway, and that it does exit when there is no unfinished work remaining.

The work class is copy-constructible so that it may be used as a data member in a handler class. It is not assignable.
